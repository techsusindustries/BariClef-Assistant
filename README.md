# BariClef Assistant
A basic minecraft modpack for fabric 1.21 that makes it easier to add automated bots to your world

## Usage Guide
This modpack was created to make it easier to create and deploy Baritone + Autoclef based bots.
In order to deploy a bot, follow the following instructions:
1. Download the latest releases .mrpack file
2. Download and login to the modrinth app if you haven already at https://modrinth.com
